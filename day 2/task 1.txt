Document object:
1.The document object represents a web page that is loaded in the browser. 
2.By accessing the document object, we can access the element in the HTML page. 
3.It is loaded inside the window.It is the object of window property.
4.The document object can be accessed with a window.document or just document.
5.With the help of document objects, we can add dynamic content to our web page.
6.All the tags, elements with attributes in HTML are part of the document.
7.We can access the document from a window using the window. document.
8.The document is part of BOM (browser object model) and dom (Document object model).

Syntax for property of document:
document.property_name;

Syntax for method of document:
document.method_name;

Window object:
1.The window object is the topmost object of the DOM hierarchy.
2.It represents a browser window or frame that displays the contents of the webpage.
3.It is the very first object that is loaded in the browser.
4.It is the object of the browser.
5.Whenever a window appears on the screen to display the contents of the document, the window object is created. 
6.Global objects, functions, and variables of JavaScript are members of the window object.
7.We can access the window from the window only. i.e. window.window.
8.The window is part of BOM, not DOM.

Syntax for property of window :
window.property_name;

Syntax for method of window:
window.method_name;
